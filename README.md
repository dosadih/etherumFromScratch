EtherumFromScratch
===============================

version number: 0.0.1
author: Damien Pretet

Overview
--------

Introduction to Etherum with Python 2.7

Installation / Usage
--------------------

To install use pip:

    $ pip install EtherumFromScratch


Or clone the repo:

    $ git clone https://github.com/dosadih/EtherumFromScratch.git
    $ python setup.py install
